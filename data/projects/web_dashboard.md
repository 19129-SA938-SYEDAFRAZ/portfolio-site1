# Web Analytics Dashboard

**Overview:**  
Built a web-based analytics dashboard for data visualization and monitoring. Focused on performance and responsive design.

**Tech Stack:**  
React, JavaScript, FastAPI, Chart.js

**Key Features:**
- Real-time data visualization  
- Modular component-based UI  
- Integration with REST API backend  

**Challenges Faced:**  
Data normalization and rendering optimization.  
Managing API rate limits for real-time updates.

**Outcome:**  
Improved understanding of frontend-backend coordination and data-driven UI design.
