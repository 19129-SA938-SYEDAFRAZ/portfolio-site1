# AI-Powered Portfolio Website

**Overview:**  
This is an interactive personal portfolio website that uses AI to allow visitors to chat and learn about my background, projects, and experience. The site integrates a RAG (Retrieval-Augmented Generation) system that fetches relevant information from local documents and project data.

**Tech Stack:**  
Next.js, Tailwind CSS, FastAPI, ChromaDB, LangChain, Hugging Face Inference API

**Key Features:**
- ChatGPT-style Q&A about my projects and timeline  
- Dynamic knowledge graph visualization  
- Interactive project explorer  
- Conversational timeline

**Challenges Faced:**  
Integrating RAG efficiently and designing a maintainable modular structure.  
Ensuring that AI responses stay relevant to my personal data context.

**Outcome:**  
Successfully deployed a lightweight, open-source AI-powered portfolio without relying on paid APIs.
