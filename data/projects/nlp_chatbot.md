# NLP Chatbot Project

**Overview:**  
Developed a chatbot capable of answering domain-specific questions using Hugging Face Transformers. The model was fine-tuned for context-aware dialogue and deployed with FastAPI.

**Tech Stack:**  
Python, FastAPI, Hugging Face Transformers, Sentence Transformers

**Key Features:**
- Context-aware conversational engine  
- Preprocessing pipeline for text data  
- Integration with RESTful API  

**Challenges Faced:**  
Tuning model parameters for accuracy and latency balance.  
Handling conversational state management.

**Outcome:**  
Learned how to combine NLP concepts with practical API development.
